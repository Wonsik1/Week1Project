# 계산기 만들기
---
## 프로젝트 개요
---
> 계산기를 구현하는 프로젝트입니다. 단일 클래스에서 연산과 계산을 모두 처리하는 방식에서 연산 클래스를 분리해 코드를 발전시켰습니다.
--- 
단계별 기능 설명
---
## Lv1. 계산기 구현

>- 간단한 연산이 가능한 계산기 구현.
>- 덧셈, 뺄셈, 곱셈, 나눗셈 연산이 가능.


## Lv2. 나머지 연산 기능 추가

>- 나머지 연산이 가능하도록 코드 추가.
>- 오류가 날 수 있는 예외처리 상황 구현.

## Lv3. 클래스 분리

>- 계산기 클래스와 연산 클래스를 분리.
>- 단일 책임 원칙에 의해 한 클래스가 하나의 역할만 담당하게 변경.
